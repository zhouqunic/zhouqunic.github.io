Samba介绍

这套 SMB软件能够让同一局域网内的 Unix 与 window 互相的分享数据！

安装Samba

==标注：Samab服务开启之前需要关闭两个服务，iptables防火墙（如果你熟悉可以不关闭，放行smb的端口即可，SAMBA服务TCP端口139,445 UDP端口 137,138）；selinux服务。==

samba软件结构

不需要账号密码访问的共享(security = share )

smb.conf配置文件分为两大类，一个全局设置参数，一个是文件共享设置参数

  • 查看共享文件权限

  • 分配共享文件最大权限(window用户可以创建或上传文件到共享目录)

  • 设置smb.conf配置文件

  • 检查smb.conf配置文件是否有语法错误

  • 启动samba服务

  • 查看samba端口，TCP端口139,445 UDP端口 137,138

  • 测试

Windows系统操作：

在win系统打开运行输入 \\[samba server IP] 直接访问，不需要输入任何用户和密码

  • 在windows上访问，发现可以登陆到samba服务器并且能够看到共享文件夹，但是无法进入该文件夹

需要输入账号和密码访问的共享(security = user)

创建samba用户之前必须先创建系统用户，系统用户和samba用户名一样，但密码可以设置成不一样

  • 创建系统用户test01

  • 把系统用户test01添加为samba用户并设置samba用户登录密码,并且把用户信息写入smbpasswd文件中

  • 列出Samba用户列表

  • 删除samba服务中的某个用户

  • 设置smb.conf配置文件

  • 启动samba

  • 测试

Windows系统操作：

在win系统打开运行输入 \\[samba server IP] 直接访问，需要输入用户和smb密码登录

linux客户端挂载samba共享目录

  • 需要注意的是挂载端也需要安装cifs相关的文件系统包

  • Linux开机自动挂载samba共享

  • 新增密码文件,重启验证一下

  • 如果采用免密码登录的方式
